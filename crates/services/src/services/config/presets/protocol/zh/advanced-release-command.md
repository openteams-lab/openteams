description: 高级发布协同模板，覆盖版本范围、质量校验、发布材料和上线监测。
workflow_steps:
- title: 版本范围与风险
  description: 确认纳入变更、负责人、依赖、回滚路径和上线风险。
- title: 质量校验
  description: 验证验收标准、关键回归、发布阻断项和签核证据。
- title: 发布材料
  description: 准备发布说明、操作指引、客户沟通和支持交接信息。
- title: 上线监测与复盘
  description: 跟踪上线健康度、事件、决策和发布后的后续负责人。
---

用一个负责的指挥中心协调发布就绪。
- 协调员负责范围、时间线、风险状态和上线/不上线决策。
- 质量验证根据验收标准和关键回归证明构建产物。
- 文档准备发布说明、操作指引和面向客户的变更。
- 团队以发布证据、监测记录、事件和后续负责人收尾。
