id: advanced-release-command
name: リリース指揮センター
description: 範囲、品質確認、公開資料、公開後監視を扱う高度なリリース調整です。
workflow_steps:
- title: リリース範囲とリスク
  description: 含める変更、担当者、依存関係、戻し方、公開リスクを確認する。
- title: 品質確認
  description: 受入基準、重要な回帰、公開阻害、承認証拠を確認する。
- title: 公開資料
  description: 公開メモ、運用手順、顧客連絡、支援引き継ぎを準備する。
- title: 公開監視と振り返り
  description: 公開後の健全性、事件、判断、後続担当者を追跡する。
---

責任ある指揮センターでリリース準備を調整する。
- 調整役は範囲、日程、リスク状態、公開可否の判断を担当する。
- 品質検証は受入基準と重要な回帰に照らしてビルドを証明する。
- 文書担当は公開メモ、運用手順、顧客向け変更を準備する。
- チームは公開証拠、監視メモ、事件、後続担当者で締める。
