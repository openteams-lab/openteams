## 支持的系统
- Windows
- macOS（Intel 和 Apple Silicon）
- Linux

## 前提条件
开始使用 openteams 前，请确保您有大模型的访问权限。您可以参考[模型供应商配置文档](/zh-Hans/custom-providers)来给openteams配置API密钥。

如果您已经在使用或者计划使用其他编程代理，请确保它们已正确安装并认证。目前openteams支持10种[编程代理](/zh-Hans/supported-coding-agents)，以下是常用代理的安装方法：
| 代理 | 安装命令 |
|-------|-------------|
| [Claude Code](https://docs.anthropic.com/en/docs/claude-code) | `npm install -g @anthropic-ai/claude-code` |
| [Gemini CLI](https://github.com/google-gemini/gemini-cli) | `npm install -g @google/gemini-cli` |
| [Codex](https://github.com/openai/codex) | `npm install -g @openai/codex` |
| [OpenCode](https://github.com/anomalyco/opencode) | `npm install -g opencode-ai@latest` |

<Tip>
您可以参考各编程代理的官方文档进行安装。
</Tip>

    **方式一：使用 npx**

    ```bash
    npx openteams-web
    ```

    **方式二：桌面安装包（推荐）**

    下载最新安装程序（`.msi`）：

    <a href="https://github.com/openteams-lab/openteams/releases/latest" target="_blank" rel="noopener noreferrer">

    </a>

    安装完成后，直接启动 **openteams**。
  </Tab>

  <Tab title="macOS">
    **使用 npx 安装启动**

  </Tab>

  <Tab title="Linux">
    **方式一：使用 npx（推荐）**

    ```bash
    npx openteams-web
    ```

    **方式二：桌面安装包**

    从 Releases 下载 Linux 安装包：
    <a href="https://github.com/openteams-lab/openteams/releases/latest" target="_blank" rel="noopener noreferrer">
      <img noZoom src="https://img.shields.io/badge/下载-Linux-FCC624?style=for-the-badge&logo=linux&logoColor=black" alt="下载 Linux 版" />

## 快速入门

<Steps>

<Step title="启动 openteams">
  选择一种启动方式：

  **方式一：通过桌面安装包启动**
  从应用菜单中打开已安装的 **openteams**。

  **方式二：通过 npx 启动**
  ```bash
  npx openteams-web
  ```
  <Note>
  系统会默认使用openteams-cli作为基础代理，为了正常访问模型，您需要在[模型供应商配置](/zh-Hans/custom-providers)中配置API。
    您也可用使用其他Agent，详情请参考[支持的编程代理](/zh-Hans/supported-coding-agents)，使用前请确保已正确安装并认证。
  </Note>
</Step>

<Step title="创建会话">
  启动后，点击"新建会话"按钮，输入会话名称并选择配置一个默认工作区路径。
  这不是强制要求，您也可以直接跳过填写，那么会话标题和默认工作区将会保持为默认值。

  ![创建会话](../images/zh/create_session_zh.png)
  
  <Note>
  默认工作区是此会话中所有AI成员的默认工作路径，您可以在团队配置中为每个成员单独设置工作区路径，并且优先级高于会话默认工作区。
  </Note>
</Step>

<Step title="创建团队">
  点击"立即创建团队"按钮，选择一个适合您任务的预设团队，或者添加预设AI成员和自定义AI成员，组建自己的专属团队。
  ![创建团队](../images/zh/createteam_zh.png)

  以全栈交付团队为例，团队包含了项目经理、前端开发、后端开发、测试工程师、UI设计师等角色，能够协作完成一个完整的项目交付。
  您需要为每位成员配置他们的模型、工作路径和技能，当然您同样可以保持默认配置，直接使用他们。
  ![团队成员配置](../images/zh/team_members_zh.png)
</Step>

<Step title="开始协作">
  您已经成功召唤了您的AI团队成员，现在让他们开始工作吧！可以直接在输入框中输入消息并通过@发送给团队成员。
  ![开始协作](../images/zh/start_collabrate_zh.png)
  例如，您可以@项目经理发送消息来分配任务，或者@前端开发请求他们编写一个登录页面的代码。

</Step>
</Steps>

<Tip>
恭喜您已经基本掌握了openteams的使用方法!

您可以根据自己的需求创建不同类型的团队来完成各种任务。更多功能请继续阅读后续章节。
</Tip>
