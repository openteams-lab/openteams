## 内置AI成员
openteams不再认为AI只是一个工具，而是将AI作为团队成员来管理，AI成员是openteams的基本组成单元。

我们相信随着AI能力的提升，AI能独立完成更多的工作，因此在openteams中，每个AI成员都有自己的角色职责、专属技能、工作区，您可以给他们配置适合其职责的模型，
让他们可以更好地在团队中协作中完成任务。
![内置AI成员](../images/builtin_members_zh.png)
我们为openteams预设了160个内置AI成员，涵盖了开发、内容创作、营销、运营、数据分析等多个领域的常见角色，
您可以直接使用这些内置成员来组建团队，也可以在此基础上进行修改，或者添加自定义AI成员来满足您的需求。
