<Steps>
<Step title="Run OpenCode">
  ```bash
  npx -y opencode-ai
  ```
</Step>

<Step title="Follow the login instructions">
  Complete the authentication flow as prompted. For more details, see the [OpenCode GitHub page](https://github.com/sst/opencode).
</Step>

<Step title="Start openteams">
  Once authenticated, launch openteams:

  **Option 1: Run with npx**
  ```bash
  npx openteams
  ```

  **Option 2: Use Desktop Installer**
  Launch the installed desktop app directly. See [Getting Started](/getting-started) for installer instructions.

  You can now select OpenCode when creating task attempts.
</Step>
</Steps>
