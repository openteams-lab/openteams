<script id="openteams-browser-language-redirect">{`(function(){
  if (typeof window === 'undefined') return;
  var path = window.location.pathname.replace(/\/+$/, '') || '/';
  var basePath = path === '/docs' || path.startsWith('/docs/') ? '/docs' : '';
  var relativePath = basePath ? (path.slice(basePath.length) || '/') : path;
  var localePrefixes = ['zh-Hans', 'zh-Hant', 'ja', 'ko', 'fr'];
  if (localePrefixes.some(function(prefix) {
    return relativePath === '/' + prefix || relativePath.startsWith('/' + prefix + '/');
  })) return;
  try {
    if (document.referrer) {
      var referrerUrl = new URL(document.referrer);
      if (referrerUrl.origin === window.location.origin) return;
    }
  } catch (error) {}
  var candidates = Array.isArray(navigator.languages) && navigator.languages.length
    ? navigator.languages
    : [navigator.language || 'en'];
  var matched = 'en';
  for (var i = 0; i < candidates.length; i += 1) {
    var normalized = String(candidates[i] || '').toLowerCase();
    if (normalized.indexOf('zh-hant') === 0 || normalized.indexOf('zh-tw') === 0 || normalized.indexOf('zh-hk') === 0 || normalized.indexOf('zh-mo') === 0) {
      matched = 'zh-Hant';
      break;
    }
    if (normalized.indexOf('zh') === 0) {
      matched = 'zh-Hans';
      break;
    }
    if (normalized.indexOf('ja') === 0) {
      matched = 'ja';
      break;
    }
    if (normalized.indexOf('ko') === 0) {
      matched = 'ko';
      break;
    }
    if (normalized.indexOf('fr') === 0) {
      matched = 'fr';
      break;
    }
  }
  if (matched === 'en') return;
  var targetRelativePath = relativePath === '/' ? '/' + matched : '/' + matched + relativePath;
  var target = basePath + targetRelativePath + window.location.search + window.location.hash;
  if (target !== window.location.pathname + window.location.search + window.location.hash) {
    window.location.replace(target);
  }
})();`}</script>


## Supported Systems

OpenTeams runs on the following platforms:

- macOS (Intel and Apple Silicon)
- Linux
- Windows

## Prerequisites

Before installing OpenTeams, ensure you have at least one supported AI agent installed:

| Agent | Installation |
|-------|-------------|
| [Claude Code](https://docs.anthropic.com/en/docs/claude-code) | `npm install -g @anthropic-ai/claude-code` |
| [Gemini CLI](https://github.com/google-gemini/gemini-cli) | `npm install -g @google/gemini-cli` |
| [Codex](https://github.com/openai/codex) | `npm install -g @openai/codex` |
| [QWen Coder](https://qwenlm.github.io/qwen-code-docs/en/users/overview/) | `npm install -g @qwen-code/qwen-code@latest` |

<Tip>
You can install other agents by following their official documentation.
</Tip>

## Quick Start

<Steps>
<Step title="Install a coding agent">
  Make sure you have at least one [supported coding agent](/supported-coding-agents) installed and authenticated on your system.
</Step>

<Step title="Launch OpenTeams">
  Choose one startup method:

  **Option 2: Run with npx**
  ```bash
  npx openteams
  ```

  **Option 1: Use Desktop Installer**
  Open the installed **OpenTeams** app from your system launcher.

  <Check>
  The application binds to a random free port and prints the URL in the terminal.
  </Check>
</Step>

<Step title="Create a chat session">
  Click "Create Session" to start a new group chat. This is your workspace for AI collaboration.
</Step>

<Step title="Add AI members">
  In the right sidebar, click "Add AI member" to add agents to your session. Configure each member with:
  - A unique @mention name
  - Select the agent type (e.g., Claude Code, Codex)
  - Set a system prompt to define their role
  - Specify their workspace path
</Step>

<Step title="Start collaborating">
  Use @mentions to assign tasks to your AI team members. They will automatically receive context and can communicate with each other!
</Step>
</Steps>

<Tip>
To use a fixed port, specify the `PORT` environment variable: `PORT=8080 npx openteams`
</Tip>
