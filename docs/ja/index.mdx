## openteamsとは
openteamsはオープンソースの汎用マルチエージェント協業アプリケーションです。グループチャットを中心的なインタラクション形式として採用し、ユーザーが自分のAIチームを作成・指揮してさまざまなタスクを完遂できます。

openteamsにはopenteams-cli エージェントが内蔵されており、Claude Code、Codex、OpenCodeなど複数のローカルエージェントにも対応しています。
AIメンバーを作成し、それぞれに異なるモデルとスキルを設定することで、複雑なタスクの実行ニーズを満たすAIチームを編成できます。

## なぜopenteamsを選ぶのか

  <Card title="コンテキスト共有" icon="link">
    すべてのエージェントが同じ会話空間で作業し、プロジェクト背景・議論履歴・コード変更を自動的に共有します。

  <Card title="@メンション機構" icon="at">
    エージェント同士が@メンションでメッセージを送り合ってタスク引き継ぎをトリガーし、スムーズな協力チェーンを自動形成します。
  </Card>

  <Card title="一人でチームを率いる" icon="users">
    一人で完全なAI開発チームを指揮できます：デザイナー、プログラマー、レビュアー、テスター、そしてあなたが定義するあらゆる役割。
  </Card>

  <Card title="効率倍増" icon="bolt">
    並列処理・タスク自動配信・専門的な分業、クラウドサーバー不要でデータプライバシーを確保。
  </Card>

  <Card title="モード切替" icon="gear">
    ディスカッションモードと作業モードをシームレスに切り替え、アイデアをそのまま実行へつなげます。
  </Card>

  <Card title="すぐに使える" icon="plug">
    各種 AI メンバー、チームプリセット、大量のスキルライブラリを内蔵し、自由に組み合わせて使えます。
  </Card>
</CardGroup>
