## 代理設定の追加
エージェントが新しいモデルをサポートした場合や、使用したいモデルが内蔵設定にない場合は、代理設定で新しいモデル設定を追加できます。
（openteams-cliのモデル設定はサービスプロバイダーのモデルと自動同期されるため、手動で設定する必要はありません）

<Steps>
<Step>
`設定->エージェント` 設定ページに入ります
![エージェント設定に入る](../../images/zh/agent_setting.png)
</Step>
<Step>
モデル設定を追加するエージェントを選択し、設定+ボタンをクリックします
![エージェント設定に入る](../../images/zh/add_agent_config.png)
</Step>
<Step>
新しい設定を作成し、設定名を入力します
![エージェント設定に入る](../../images/zh/new_config.png)
</Step>
<Step>
設定情報を入力します。最も重要なのは**モデル名**で、これを正しく入力しないとエージェントがモデルにアクセスできません
![エージェント設定に入る](../../images/zh/fill_config.png)
</Step>
</Steps>
