<Steps>
<Step title="OpenCodeを実行">
  ```bash
  npx -y opencode-ai
  ```
</Step>

<Step title="ログイン手順に従う">
  プロンプトに従って認証フローを完了します。詳細は[OpenCode GitHubページ](https://github.com/sst/opencode)を参照してください。
</Step>

<Step title="openteamsを起動">
  認証完了後、openteamsを起動：

  **方法1：npxで起動**
  ```bash
  npx openteams
  ```

  **方法2：Desktop Installerで起動**
  インストール済みのデスクトップアプリを直接起動できます。インストーラー手順は[はじめに](/ja/getting-started)を参照してください。

  タスク試行を作成する際にOpenCodeを選択できるようになりました。
</Step>
</Steps>
