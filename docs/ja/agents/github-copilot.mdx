<Steps>
<Step title="GitHub Copilotを実行">
  ```bash
  npx -y @github/copilot
  ```
</Step>

<Step title="ログイン手順に従う">
  プロンプトが表示されたら、画面の指示に従って `/login` コマンドで認証します。詳細は [GitHub Copilot CLI ドキュメント](https://docs.github.com/en/copilot/how-tos/use-copilot-agents/use-copilot-cli) を参照してください。
</Step>

</Steps>
