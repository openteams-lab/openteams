## 自訂設定目錄

預設情況下，Codex 將其設定和會話資料儲存在 `~/.codex`。如果您使用 `CODEX_HOME` 環境變數設定了自訂位置，openteams 將自動偵測並使用該位置。

```bash
# 範例：使用專案特定的 Codex 設定
export CODEX_HOME=/path/to/custom/codex
npx openteams
```

這對以下情況很有用：
- 專案特定的 Codex 設定檔
- 分離工作和個人設定
- 自訂自動化設定
