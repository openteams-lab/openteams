<Steps>
<Step title="執行 GitHub Copilot">
  ```bash
  npx -y @github/copilot
  ```
</Step>

<Step title="按照登入說明操作">
  出現提示時，按照螢幕上的說明使用 `/login` 指令進行認證。更多詳情請參閱 [GitHub Copilot CLI 文件](https://docs.github.com/en/copilot/how-tos/use-copilot-agents/use-copilot-cli)。
</Step>

</Steps>
