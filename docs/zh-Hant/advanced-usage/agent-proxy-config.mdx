## 新增代理設定
您的代理 Agent 可能支援了新模型，或者您想使用的模型在內建設定中沒有，因此您可以在代理設定中新增新的模型設定。
（openteams-cli 的模型設定會自動同步服務供應商中的模型，您無需手動設定）

<Steps>
<Step>
進入 `設定 -> 代理` 設定頁面
![進入代理設定](../../images/zh/agent_setting.png)
</Step>
<Step>
選擇您要新增模型設定的代理，點擊設定 + 按鈕
![進入代理設定](../../images/zh/add_agent_config.png)
</Step>
<Step>
新建一個設定，輸入設定名稱
![進入代理設定](../../images/zh/new_config.png)
</Step>
<Step>
填寫好設定資訊，最重要的就是**模型名稱**，這個必須填寫正確否則 Agent 無法存取模型
![進入代理設定](../../images/zh/fill_config.png)
</Step>
</Steps>
