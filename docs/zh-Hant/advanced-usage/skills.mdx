## 什麼是技能
每一個技能都封裝了一類具體能力，例如程式碼產生、資訊檢索、資料處理、檔案操作或呼叫外部 API。
技能通常包含明確的輸入輸出定義、執行邏輯以及與環境互動的方式，使智能體能夠在不同場景下穩定地呼叫和複用這些能力。

透過為智能體設定不同的技能，可以靈活地定義其「能做什麼」和「如何做」。
多個技能可以組合使用，形成更複雜的工作流程，從而支援跨任務、多步驟的問題解決。

## 安裝技能
openteams 中提供了技能庫，您可以在軟體中為您的 Agent 代理安裝技能，軟體也會讀取當前代理已安裝的技能列表。
在市場技能列表中搜尋您想要安裝的技能，選擇要安裝的 Agent 代理，點擊安裝即可。

## Agent 代理層面開啟技能
在已安裝技能列表中為 Agent 代理開啟技能（部分 Agent 自身不支援開關技能，如 ClaudeCode）
![](../../images/zh/install_skill.png)

## 賦予 AI 成員技能
需要給團隊中的 AI 成員設定對應的技能後，成員才允許使用此技能。可以在成員屬性設定中的技能列表中選擇已安裝的技能賦予成員。
![賦予 AI 成員技能](../../images/zh/config_member_skill.png)
