<Steps>
<Step title="Exécuter Amp">
  ```bash
  npx -y @sourcegraph/amp
  ```
</Step>

<Step title="Suivre les instructions de connexion">
  Complétez le flux d'authentification comme indiqué. Pour plus de détails, voir le [Manuel Amp](https://ampcode.com/manual#install).
</Step>

<Step title="Démarrer openteams">
  Une fois authentifié, lancez openteams :

  **Option 1 : Démarrer avec npx**
  ```bash
  npx openteams
  ```

  **Option 2 : Utiliser l'installateur Desktop**
  Lancez directement l'application desktop installée. Consultez le [guide de démarrage](/fr/getting-started) pour les instructions d'installation.


  Vous pouvez maintenant sélectionner Amp lors de la création de tentatives de tâches.
</Step>
</Steps>
