Les membres IA sont les unités de base d'openteams et le cœur de toute équipe IA. Chaque membre IA dispose de capacités d'agent indépendantes, peut monter des compétences et appeler des outils.
Voici comment gérer et utiliser les membres IA dans openteams.

Dans une session de chat de groupe, vous pouvez ajouter un ou plusieurs membres IA et les diriger pour collaborer sur des tâches. La collaboration peut prendre la forme de tâches parallèles totalement indépendantes ou d'une exécution séquentielle avec retour — tout dépend de votre configuration.

> À mesure que les capacités des grands modèles s'améliorent, les membres IA individuels deviendront également plus puissants. Pour obtenir davantage de gains d'efficacité, la collaboration entre plusieurs membres IA est incontournable. La façon dont les membres IA collaborent mieux ensemble est donc une question fondamentale.
>
> openteams continuera à explorer cette question jusqu'à trouver sa propre réponse.

## Ajouter et modifier des membres IA
La section [Gérer l'équipe](/fr/advanced-usage/create-team) explique comment ajouter des membres IA personnalisés — nous n'y revenons pas ici.
Nous vous expliquons principalement comment modifier les informations d'un membre IA. Commencez par cliquer sur le bouton **Modifier** dans la carte du membre.
![Modifier un membre](../../images/zh/edit_member.png)

Vous pouvez redéfinir le nom, le modèle, les instructions et les compétences utilisées par le membre IA.
<Note>
Le chemin de l'espace de travail ne peut pas être modifié après l'ajout. Vous devez d'abord supprimer le membre IA puis le ré-ajouter.
</Note>

## Espace de travail du membre IA
L'espace de travail du membre est un panneau latéral qui comprend actuellement l'historique des messages du membre et les journaux d'exécution. Nous prévoyons de supporter l'ouverture du chemin de l'espace de travail avec des outils comme `VS Code` et `Cursor` dans la version 0.3.12.
![Tiroir de l'espace de travail](../../images/zh/workspace.png)

Vous pouvez consulter les journaux d'erreurs d'exécution dans l'espace de travail pour faciliter le diagnostic des erreurs d'agent.

## Ajouter un membre IA prédéfini
Si vous souhaitez sauvegarder de manière permanente un membre IA utilisé dans une session pour l'utiliser dans d'autres sessions, vous pouvez l'ajouter ici, l'enregistrer comme préréglage et l'importer directement lors des prochaines utilisations.
La configuration est identique à celle d'un membre IA personnalisé — vous pouvez copier-coller directement.
![Ajouter un membre IA](../../images/zh/add_memeber.png)
