## Ajouter une configuration d'agent

Si votre agent prend en charge un nouveau modele, ou si le modele que vous souhaitez utiliser n'apparait pas dans la configuration integree, vous pouvez ajouter une nouvelle configuration de modele au niveau de l'agent.

(La configuration des modeles dans openteams-cli se synchronise automatiquement avec les modeles du fournisseur de service. Vous n'avez donc pas besoin de les saisir manuellement.)

<Steps>
<Step>
Ouvrez la page `Parametres -> Agents`.

![Acceder aux parametres des agents](../../images/zh/agent_setting.png)
</Step>

<Step>
Selectionnez l'agent auquel vous voulez ajouter une configuration de modele, puis cliquez sur le bouton de configuration `+`.

![Ajouter une configuration d'agent](../../images/zh/add_agent_config.png)
</Step>

<Step>
Creez une nouvelle configuration et saisissez son nom.

![Creer une nouvelle configuration](../../images/zh/new_config.png)
</Step>

<Step>
Renseignez les informations de configuration. Le champ le plus important est **le nom du modele** : il doit etre exact, sinon l'agent ne pourra pas acceder au modele.

![Renseigner la configuration](../../images/zh/fill_config.png)
</Step>
</Steps>
