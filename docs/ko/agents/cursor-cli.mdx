<Steps>
<Step title="Cursor Agent CLI 설치">
  ```bash
  curl https://cursor.com/install -fsS | bash
  ```

  `cursor-agent --version`으로 설치를 확인합니다. 자세한 내용은 [공식 설치 가이드](https://docs.cursor.com/en/cli/installation)를 참조하세요.
</Step>

<Step title="로그인 지침 따르기">
  `cursor-agent login`으로 로그인합니다 (브라우저가 열립니다). `CURSOR_API_KEY` 환경 변수를 설정할 수도 있습니다. 전체 지침: https://docs.cursor.com/en/cli/reference/authentication
</Step>

</Steps>
