## 사용자 정의 구성 디렉토리

기본적으로 Codex는 구성 및 세션 데이터를 `~/.codex`에 저장합니다. `CODEX_HOME` 환경 변수로 사용자 정의 위치를 구성한 경우 openteams이 자동으로 해당 위치를 감지하고 사용합니다.

```bash
# 예: 프로젝트별 Codex 구성 사용
export CODEX_HOME=/path/to/custom/codex
npx openteams
```

이것은 다음과 같은 경우에 유용합니다:
- 프로젝트별 Codex 구성 파일
- 업무와 개인 구성 분리
- 사용자 정의 자동화 설정
