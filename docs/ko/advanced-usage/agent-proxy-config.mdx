## 에이전트 구성 추가
에이전트가 새 모델을 지원하거나 사용하려는 모델이 내장 구성에 없는 경우, 에이전트 구성에서 새 모델 구성을 추가할 수 있습니다.
(openteams-cli의 모델 구성은 서비스 공급자의 모델과 자동으로 동기화되므로 수동으로 구성할 필요가 없습니다.)

<Steps>
<Step>
`설정->에이전트` 설정 페이지로 이동합니다.
![에이전트 설정 진입](../../images/zh/agent_setting.png)
</Step>
<Step>
모델 구성을 추가할 에이전트를 선택하고 구성+ 버튼을 클릭합니다.
![에이전트 설정 진입](../../images/zh/add_agent_config.png)
</Step>
<Step>
새 구성을 만들고 구성 이름을 입력합니다.
![에이전트 설정 진입](../../images/zh/new_config.png)
</Step>
<Step>
구성 정보를 입력합니다. 가장 중요한 것은 **모델 이름**으로, 이를 정확히 입력하지 않으면 에이전트가 모델에 접근할 수 없습니다.
![에이전트 설정 진입](../../images/zh/fill_config.png)
</Step>
</Steps>
