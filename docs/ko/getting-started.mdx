## 지원 시스템
- Windows
- macOS (Intel 및 Apple Silicon)
- Linux

## 사전 요구사항
openteams을 사용하기 전에 대형 언어 모델에 대한 접근 권한이 있는지 확인하세요. [모델 공급자 구성 문서](/ko/custom-providers)를 참고하여 openteams에 API 키를 설정할 수 있습니다.

다른 프로그래밍 에이전트를 이미 사용 중이거나 사용할 계획이라면, 해당 에이전트가 올바르게 설치되고 인증되었는지 확인하세요. 현재 openteams은 10가지 [프로그래밍 에이전트](/ko/supported-coding-agents)를 지원하며, 주요 에이전트의 설치 방법은 다음과 같습니다:

| 에이전트 | 설치 명령어 |
|-------|-------------|
| [Claude Code](https://docs.anthropic.com/en/docs/claude-code) | `npm install -g @anthropic-ai/claude-code` |
| [Gemini CLI](https://github.com/google-gemini/gemini-cli) | `npm install -g @google/gemini-cli` |
| [Codex](https://github.com/openai/codex) | `npm install -g @openai/codex` |
| [OpenCode](https://github.com/anomalyco/opencode) | `npm install -g opencode-ai@latest` |

<Tip>
각 프로그래밍 에이전트의 공식 문서를 참고하여 설치할 수 있습니다.
</Tip>

## 빠른 시작

<Steps>

<Step title="openteams 실행">
  실행 방식을 선택하세요:

  **방법 1: 데스크톱 설치 패키지로 실행**
  앱 메뉴에서 설치된 **openteams**을 엽니다.

  **방법 2: npx로 실행**
  ```bash
  npx openteams-web
  ```
  <Note>
  시스템은 기본적으로 openteams-cli를 기본 에이전트로 사용합니다. 모델에 정상적으로 접근하려면 [모델 공급자 구성](/ko/custom-providers)에서 API를 설정해야 합니다.
    다른 에이전트를 사용할 수도 있습니다. 자세한 내용은 [지원되는 프로그래밍 에이전트](/ko/supported-coding-agents)를 참고하시고, 사용 전에 올바르게 설치 및 인증되었는지 확인하세요.
  </Note>
</Step>

<Step title="세션 생성">
  실행 후 "새 세션" 버튼을 클릭하고, 세션 이름을 입력한 후 기본 작업 공간 경로를 선택합니다.
  이는 필수 사항이 아니므로, 그냥 건너뛰어도 됩니다. 이 경우 세션 제목과 기본 작업 공간은 기본값으로 유지됩니다.

  ![세션 생성](../images/en/create_session_zh.png)

  <Note>
  기본 작업 공간은 이 세션의 모든 AI 멤버에 대한 기본 작업 경로입니다. 팀 구성에서 각 멤버별로 개별 작업 공간 경로를 설정할 수 있으며, 이는 세션 기본 작업 공간보다 우선순위가 높습니다.
  </Note>
</Step>

<Step title="팀 생성">
  "지금 팀 만들기" 버튼을 클릭하고, 작업에 적합한 사전 설정 팀을 선택하거나 사전 설정 AI 멤버와 사용자 정의 AI 멤버를 추가하여 나만의 전용 팀을 구성합니다.
  ![팀 생성](../images/en/createteam_zh.png)

  풀스택 개발 팀을 예로 들면, 프로젝트 매니저, 프론트엔드 개발자, 백엔드 개발자, QA 엔지니어, UI 디자이너 등의 역할이 포함되어 있어 완전한 프로젝트 개발을 협업으로 완수할 수 있습니다.
  각 멤버에게 모델, 작업 경로, 스킬을 구성해야 합니다. 물론 기본 구성을 유지하고 바로 사용할 수도 있습니다.
  ![팀 멤버 구성](../images/en/team_members.png)
</Step>

<Step title="협업 시작">
  AI 팀 멤버를 성공적으로 소환했습니다. 이제 그들이 일을 시작하게 하세요! 입력창에 메시지를 입력하고 @를 사용하여 팀 멤버에게 전송할 수 있습니다.
  ![협업 시작](../images/en/start_collabrate.png)
  예를 들어, @프로젝트매니저에게 메시지를 보내 작업을 할당하거나, @프론트엔드에게 로그인 페이지 코드 작성을 요청할 수 있습니다.

</Step>
</Steps>

<Tip>
openteams의 기본 사용법을 익히셨습니다!

필요에 따라 다양한 유형의 팀을 만들어 여러 작업을 수행할 수 있습니다. 더 많은 기능은 다음 섹션을 계속 읽어보세요.
</Tip>
